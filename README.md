**Easy Forma Maker** is an open-source tool designed to automate the forma design process for professional press printing. 

Organizing a book with hundreds of pages into a print-ready forma layout is often a "nightmare" that takes hours of manual work. This app simplifies that workflow, allowing you to generate professional layouts in seconds.

## ‚ú® Features
* **Automated Layout:** No more manual page imposition.
* **Simple Workflow:** Go from MS Word to a print-ready PDF effortlessly.
* **User-Friendly GUI:** Easy-to-navigate interface for quick processing.
* **Open Source:** Full source code available for customization and improvements.

## üöÄ How to Use
1.  **Prepare your file:** Finish editing your book in MS Word and export/save it as a **PDF**.
2.  **Run the App:** Navigate to the `/dist` directory in this repository.
3.  **Launch:** Open `Easy Forma Maker.exe`.
4.  **Upload:** Use the GUI to load your PDF file and follow the on-screen prompts to generate your forma.
